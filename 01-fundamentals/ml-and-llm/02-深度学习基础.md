# 深度学习基础

---

## 神经网络基本结构

**前馈神经网络（Feedforward）：** 输入层 → 隐藏层 × N → 输出层，每层做线性变换 + 非线性激活。

**通用近似定理：** 一个足够宽的单隐藏层网络可以近似任意连续函数。但实践中，深层网络（多隐藏层）比宽而浅的网络更高效。

---

## 激活函数（Activation Function）

| 激活函数 | 公式 | 优点 | 缺点 |
|---|---|---|---|
| Sigmoid | σ(x) = 1/(1+e⁻ˣ) | 输出在 (0,1)，适合概率 | 梯度消失（两端饱和区梯度接近 0） |
| Tanh | tanh(x) | 输出在 (-1,1)，零中心 | 同样有梯度消失问题 |
| ReLU | max(0, x) | 计算简单，缓解梯度消失 | 负区间梯度为 0（Dead ReLU） |
| Leaky ReLU | max(0.01x, x) | 解决 Dead ReLU | 需要调超参 α |
| GELU | x × Φ(x) | 平滑版 ReLU | 计算稍复杂 |
| SwiGLU | Swish(xW) × (xV) | 效果好，GLU 门控机制 | 参数量增加 |

**LLM 主流选择：** GELU（BERT、GPT）→ SwiGLU（LLaMA、PaLM，效果更好）

## 反向传播（Backpropagation）

**链式法则：** 从输出层到输入层，逐层计算损失对每个参数的梯度。

```
前向传播：x → h₁ → h₂ → ... → y → Loss
反向传播：∂L/∂y → ∂L/∂h₂ → ∂L/∂h₁ → ... → ∂L/∂W（更新权重）
```

**梯度消失（Vanishing Gradient）：** 深层网络中，梯度经过多层相乘后变得极小，底层权重几乎无法更新。
- 原因：Sigmoid/Tanh 的导数 < 1，多层连乘趋近于 0
- 解决：ReLU 激活、残差连接（ResNet）、Batch Normalization

**梯度爆炸（Exploding Gradient）：** 梯度过大，权重更新失控。
- 解决：梯度裁剪（Gradient Clipping），将梯度范数限制在阈值内

## Batch Normalization

**原理：** 在每一层的输出上做归一化（减均值、除标准差），使每层输入分布稳定。

```
BN(x) = γ × (x - μ_batch) / √(σ²_batch + ε) + β
```

- γ 和 β 是可学习参数，保留网络的表达能力
- 训练时用 mini-batch 统计量，推理时用全局均值/方差（Running Average）

**作用：**
- 加速收敛（可以用更大学习率）
- 轻微正则化效果
- 缓解梯度消失/爆炸

**Layer Normalization：** 在特征维度做归一化（而非 batch 维度），不依赖 batch size。**Transformer / LLM 的标配。**

| 方法 | 归一化维度 | 适用场景 |
|---|---|---|
| Batch Norm | batch 维度 | CNN、固定 batch size |
| Layer Norm | 特征维度 | Transformer、RNN、变长序列 |
| RMSNorm | 特征维度（只除 RMS，不减均值） | LLaMA 等现代 LLM |

## 残差连接（Residual Connection）

**核心思想：** 每层的输出 = 层的变换(x) + x（跳跃连接）

```
y = F(x) + x
```

**为什么有效：**
- 梯度可以通过跳跃连接直接回传到底层，缓解梯度消失
- 最差情况下 F(x) = 0，网络退化为恒等映射，不会比浅层网络更差
- 使训练数百层甚至数千层的网络成为可能

**Transformer 中的应用：** 每个 Attention 子层和 FFN 子层都有残差连接 + Layer Norm。

## CNN 核心概念

**卷积层（Convolution）：** 用可学习的滤波器（Kernel）在输入上滑动，提取局部特征。

**关键性质：**
- **参数共享：** 同一个 Kernel 在所有位置复用，大幅减少参数量
- **局部连接：** 每个输出只依赖局部输入区域（感受野 Receptive Field）
- **平移等变性：** 特征不因位置变化而改变

**池化层（Pooling）：** 下采样，减少空间维度。Max Pooling（取最大值）/ Average Pooling。

**经典架构演进：** LeNet → AlexNet → VGG → ResNet → EfficientNet → ViT（用 Transformer 替代 CNN）

## RNN / LSTM / GRU

**RNN：** 处理序列数据，隐状态 h_t = f(W_h × h_{t-1} + W_x × x_t)。
- 问题：长序列梯度消失/爆炸，难以捕捉长距离依赖

**LSTM（Long Short-Term Memory）：**
- 引入门控机制：遗忘门、输入门、输出门
- Cell State 作为"信息高速公路"，可以选择性地记住/遗忘信息
- 解决了 RNN 的长距离依赖问题

**GRU（Gated Recurrent Unit）：** LSTM 的简化版，合并遗忘门和输入门为更新门，参数更少，效果接近。

**现状：** Transformer 在几乎所有序列任务上超越 RNN/LSTM，但 RNN 的线性复杂度在超长序列上仍有优势（如 Mamba 等 SSM 模型）。

## 常见深度学习技巧

**权重初始化：**
- Xavier 初始化：适配 Sigmoid/Tanh，使每层输出方差一致
- Kaiming 初始化：适配 ReLU，考虑 ReLU 将一半输出置零
- 错误的初始化（如全 0）会导致对称性问题，所有神经元学到相同的东西

**混合精度训练（Mixed Precision）：**
- 前向/反向传播用 FP16/BF16（快、省显存）
- 参数更新（optimizer step）用 FP32（保证精度）
- Loss Scaling：放大 Loss 防止 FP16 下梯度下溢

**梯度累积（Gradient Accumulation）：**
- 显存不够放大 batch 时，多个 mini-batch 的梯度在本地累积，等价于更大的 batch
- 实际 batch size = micro_batch_size × accumulation_steps × num_gpus
